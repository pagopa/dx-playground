dx-playground

Dummy web app for Azure Static Web Apps.

This workspace contains only static assets and produces a plain dist folder for the shared release workflow.

Build

Zero framework runtime

The build step copies files from public to dist with no server or framework-specific export.

Deploy

Shared DX workflow

Release is delegated to the PagoPA DX reusable workflow for Azure Static Web Apps.

Scope

Deliberately minimal

This app exists only to validate the Azure Static Web Apps release path in this repository.
